# Tableau Analysis

## Objective
To visualize and analyze key healthcare cost drivers using interactive dashboards.

## Dataset Used
- Source: Cleaned healthcare dataset
- Key fields:
  - Smoker
  - BMI Category
  - Age Group
  - Charges

## Worksheets Created
1. Average Healthcare Cost by Smoker
2. Average Healthcare Cost by BMI Category
3. Average Healthcare Cost by Age Group

## Dashboard Design
- Combined multiple charts into a single dashboard
- Used consistent formatting for readability
- Focused on cost comparison across drivers

## Key Insights
- Smokers have significantly higher healthcare costs
- Obese BMI category shows the highest average charges
- Healthcare costs increase with age group

## Validation
Insights from Tableau align with results from Excel and SQL analysis.
