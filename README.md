# Healthcare Cost Drivers Analysis

## Overview
This project presents an end-to-end analysis of healthcare cost drivers using **Excel, SQL, Python, and Tableau**.  
The objective is to identify key factors influencing healthcare charges and validate insights consistently across multiple analytical tools.

The project demonstrates:
- Data cleaning and transformation
- Aggregation and statistical analysis
- Cross-tool validation of insights
- Professional documentation and reproducibility
